# Bases de Données (SQL)

Cette page sert d'aide-mémoire pour préparer, structurer et utiliser des bases de données relationnelles avec le langage SQL.

## 1. Concepts de base

Les bases de données relationnelles (SGBDR) stockent les données dans des tables liées entre elles.
Voici quelques termes essentiels :
- **Table** : Structure de base contenant des données réparties en colonnes et lignes.
- **Clé Primaire (Primary Key - PK)** : Identifiant unique d'une ligne dans une table (ex: ID).
- **Clé Étrangère (Foreign Key - FK)** : Référence à la clé primaire d'une autre table, créant ainsi une relation.
